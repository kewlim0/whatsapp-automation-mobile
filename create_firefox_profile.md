# How to Create a Firefox Profile Named "NepalWin"

## Steps:

1. **Close all Firefox windows**

2. **Open Firefox Profile Manager**:
   - Press `Win + R`
   - Type: `firefox.exe -P`
   - Press Enter

3. **Create New Profile**:
   - Click "Create Profile"
   - Click "Next"
   - Enter profile name: `NepalWin`
   - Click "Finish"

4. **Launch the new profile**:
   - Select "NepalWin" from the list
   - Uncheck "Use the selected profile without asking at startup" (if you want to keep your default profile)
   - Click "Start Firefox"

5. **Login to WhatsApp Web**:
   - Navigate to https://web.whatsapp.com
   - Scan QR code with your phone
   - Keep this session logged in

6. **Close Firefox**

Now your script will automatically detect and use the "NepalWin" profile!

## Profile Location:
Windows: `%APPDATA%\Mozilla\Firefox\Profiles\XXXXXXXX.NepalWin`

The script will automatically find it.
